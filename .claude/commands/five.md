Perform a Five Whys root cause analysis on the problem described below.

Problem: $ARGUMENTS

## Instructions

1. State the problem clearly
2. Ask "Why?" five times, each answer becoming the input for the next question
3. Identify the root cause at the deepest level
4. Propose 1-3 corrective actions that address the root cause (not symptoms)

## Output format

```
## Five Whys Analysis

**Problem**: [restate clearly]

1. Why? → [first cause]
2. Why? → [second cause]
3. Why? → [third cause]
4. Why? → [fourth cause]
5. Why? → [root cause]

**Root Cause**: [summary]

**Corrective Actions**:
1. [action] — [owner / timeline]
2. [action] — [owner / timeline]
```

If fewer than 5 whys are needed to reach the root cause, stop there and explain why.
